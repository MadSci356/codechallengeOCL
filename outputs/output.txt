usage: petsearch.py [-h] -t TYPE -l LOCATION [-j]

Simple Pet Finder using Oak City Labs API

optional arguments:
  -h, --help            show this help message and exit
  -t TYPE, --type TYPE  Animal type. eg: dog/cat/rabbit
  -l LOCATION, --location LOCATION
                        Location to search. eg: Raleigh,NC/Charleston,SC
  -j, --json            Print out JSON results instead of normal output
